#  Student Management System

A Python-based Student Management System designed to manage and analyze student marks through a simple menu-driven interface. The application allows users to view student records, calculate statistics, search for students, assign grades, and update the student list.

##  Features

- Display all student records
- Find the highest-scoring student
- Calculate average marks
- Calculate total marks
- Display students scoring above 80
- Search for a student by name
- Add a new student
- Remove a student
- Assign grades based on marks

##  Technologies Used

- Python 3
- Functions
- Lists
- Loops
- Conditional Statements

##  How to Run

1. Clone this repository.
2. Open the project folder.
3. Run the following command:

```bash
python student_management_system.py
```
